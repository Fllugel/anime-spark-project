# Anime Spark Project

Проект для аналізу датасету аніме за допомогою Apache Spark. Створює зірчасту схему даних, виконує бізнес-питання та підготовляє дані для Machine Learning моделей.

## Що робить проект

1. **Трансформація даних** - створює зірчасту схему (Star Schema) з вимірів та фактів
2. **Бізнес-аналіз** - виконує бізнес-питання для аналізу даних
3. **ML-підготовка** - готує датасети для навчання моделей регресії та класифікації

## Як запустити

### Вимоги

- Docker Desktop встановлений та запущений
- Дані датасету в папці `data/` (CSV файли)

### Запуск через Docker

**Windows:**
```batch
run_docker_windows.bat
```

**Mac/Linux:**
```bash
./run_docker_mac_linux.sh
```

Скрипти автоматично:
- Перевіряють наявність Docker образу
- Будують образ, якщо він не існує
- Запускають проект в контейнері

### Що відбувається при запуску

Після запуску з'явиться меню з вибором:

1. **Бізнес-питання** - створює зірчасту схему та виконує аналіз
2. **Регресія ML** - підготовляє датасет та запускає моделі регресії
3. **Класифікація ML** - підготовляє датасет та запускає моделі класифікації

Результати зберігаються в папці `data/results/`.

## Структура проекту

```
anime-spark-project/
├── data/                    # Дані та результати
│   ├── star_schema/         # Зірчаста схема (Parquet)
│   ├── ml_datasets/         # ML-датасети
│   └── results/             # Результати аналізу
├── transformation/          # Модулі трансформації даних
├── data_analysis/           # Модулі ML-аналізу
├── main.py                  # Головний файл з меню
├── Dockerfile               # Docker конфігурація
├── run_docker_windows.bat   # Скрипт запуску (Windows)
└── run_docker_mac_linux.sh  # Скрипт запуску (Mac/Linux)
```

## Технології

- Apache Spark 3.4.1
- PySpark
- Docker
- Python 3.9
